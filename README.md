# Biblia_Windows
Notas Generales


## Crear Entorno Python

### Cambiar Path del Proyecto
```
cd C:\Users\TuUsuario\Proyectos\MiProyecto

```
### Crear Entorno
```
python -m venv mi_entorno
```
### Activar Entorno
```
.\mi_entorno\Scripts\activate
```
### Instalar Librerias
```
pip install matplotlib
```
